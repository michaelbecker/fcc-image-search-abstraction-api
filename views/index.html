FreeCodeCamp Image Search Abstraction API Project

User Stories

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Examples

  • https://quiver-smash.glitch.me/api/imagesearch/lolcats%20funny?offset=10
  • https://quiver-smash.glitch.me/api/latest/imagesearch

Discussion

We leverage a third party API to actually perform the real image lookup, as recommended by FCC. We consume this data and reformat it to return in the format implicitly specified.

Regarding returning the search history, we save approximately the last 10 search terms in a MongoDB database. With each new query we add the new search term, then prune the DB collection to the last 10. When looking for the latest image searches, we simply query all documents in the DB, format them, and return them.
